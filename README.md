📚 Pustok – Online Book Store

Pustok is a simple e-commerce web application for browsing and purchasing books online.
It was built using ASP.NET Core MVC and demonstrates a layered MVC structure with separate services and data access.

This project was created to practice building a full-stack web application with product management, cart functionality, and database integration.

🚀 Features

Browse books and categories

Search books across the store

Add books to shopping cart

User authentication

Admin area for managing products

Responsive UI
